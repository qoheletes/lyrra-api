YouTube Domain

Audio download, Whisper transcription/translation, and sentence segmentation

The YouTube domain handles the core language-learning pipeline: it accepts a YouTube URL, downloads the audio via yt-dlp, sends it to the OpenAI Whisper API for transcription and translation to English, then persists the result to disk as JSON. On subsequent requests for the same video, the cached result is returned immediately without re-downloading or re-transcribing.

POST /youtube/translate Translate a YouTube video to English

Accepts a YouTube URL and optional Whisper settings. If the video has already been processed, returns the cached result immediately. Otherwise, downloads the audio, runs Whisper translation (always targeting English output), persists words and sentence segmentation to disk, and returns the full transcript.

Request body
FieldTypeDescription
youtube_url requiredstring (URL)Any valid YouTube URL — watch, shorts, embed, or youtu.be
model optionalstringWhisper model name. Default: whisper-1
source_language optionalstring | nullHint the source language (e.g. ko, ja). Whisper auto-detects if omitted.
include_words optionalbooleanInclude word-level timestamps in response. Default: true
Example request
{
  "youtube_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
  "model": "whisper-1",
  "source_language": null,
  "include_words": true
}
Response · 200 OK
{
  "title": "Video Title",
  "video_id": "dQw4w9WgXcQ",
  "source_language": "ko",
  "translated_text": "Full translated English transcript...",
  "words": [
    { "word": "Hello", "start": 0.0, "end": 0.48 },
    { "word": "world", "start": 0.52, "end": 1.1 }
  ]
}
200 Translated / cached 400 YouTube download failed 500 ffmpeg missing / Whisper error
GET /youtube/transcribe/{video_id}/sentences Get sentence-segmented transcript

Returns the cached transcription broken into natural sentences with start/end timestamps and per-word timing. Sentences are split on terminal punctuation (. ! ?) and words are aligned to the Whisper word-level output. Returns 404 if the video has not been transcribed yet.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID (e.g. dQw4w9WgXcQ)
Response · 200 OK

  "language": "ko",

      "end": 2.5,
      "words": [

    }
