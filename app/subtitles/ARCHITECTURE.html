Subtitles Domain

Upload, retrieve, and delete subtitle tracks (VTT/SRT) per video and language

The Subtitles domain manages subtitle files stored in the configured storage backend (local filesystem or S3). Each subtitle track is uniquely identified by its video_id + language_code pair. Files are stored under a key derived from the video ID and language code, and the public URL is persisted in the database. Retrieval issues a 302 redirect directly to the file URL.

POST /videos/{video_id}/subtitles Upload a subtitle track

Accepts a multipart/form-data upload. Stores the subtitle file in the configured storage backend, persists the track record in the database, and returns the created track metadata. Uploading for an existing language replaces the track.

Form fields
FieldTypeDescription
file requiredfileSubtitle file content (VTT or SRT)
language_code requiredstringBCP-47 language code, e.g. en, ko, ja
format optionalstringFile format: vtt (default) or srt
Example request
# curl multipart upload
curl -X POST http://localhost:8000/videos/dQw4w9WgXcQ/subtitles \
  -F "file=@subtitles_ko.vtt" \
  -F "language_code=ko" \
  -F "format=vtt"

GET /videos/{video_id}/subtitles/{language_code} Redirect to subtitle file

Looks up the subtitle track for the given video and language, then issues a 302 redirect to the file's public URL. The client follows the redirect and downloads the subtitle file directly from storage. Returns 404 if no track exists for that combination.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID
language_code requiredstringLanguage code matching a stored track
302 Redirect to file URL 404 Track not found
DELETE /videos/{video_id}/subtitles/{language_code} Delete a subtitle track

Deletes both the stored subtitle file from the storage backend and the corresponding database record. Idempotent — if the file does not exist in storage the record is still removed. Returns 204 No Content on success.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID
language_code requiredstringLanguage code of the track to delete
204 Deleted 404 Track not found
