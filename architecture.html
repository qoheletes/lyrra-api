Lyrra API Architecture

A FastAPI service for YouTube audio translation and video subtitle management. It accepts YouTube URLs, transcribes and translates speech with OpenAI Whisper, and stores structured results for fast retrieval and subtitle delivery.

FastAPI · Python 3.9+
PostgreSQL · SQLAlchemy ORM
OpenAI Whisper · Audio transcription

YouTube Domain

Audio download, Whisper transcription/translation, and sentence segmentation

The YouTube domain handles the core language-learning pipeline: it accepts a YouTube URL, downloads the audio via yt-dlp, sends it to the OpenAI Whisper API for transcription and translation to English, then persists the result to disk as JSON. On subsequent requests for the same video, the cached result is returned immediately without re-downloading or re-transcribing.

POST /youtube/translate Translate a YouTube video to English

Accepts a YouTube URL and optional Whisper settings. If the video has already been processed, returns the cached result immediately. Otherwise, downloads the audio, runs Whisper translation (always targeting English output), persists words and sentence segmentation to disk, and returns the full transcript.

Request body
FieldTypeDescription
youtube_url requiredstring (URL)Any valid YouTube URL — watch, shorts, embed, or youtu.be
model optionalstringWhisper model name. Default: whisper-1
source_language optionalstring | nullHint the source language (e.g. ko, ja). Whisper auto-detects if omitted.
include_words optionalbooleanInclude word-level timestamps in response. Default: true
Example request
{
  "youtube_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
  "model": "whisper-1",
  "source_language": null,
  "include_words": true
}
Response · 200 OK
{
  "title": "Video Title",
  "video_id": "dQw4w9WgXcQ",
  "source_language": "ko",
  "translated_text": "Full translated English transcript...",
  "words": [
    { "word": "Hello", "start": 0.0, "end": 0.48 },
    { "word": "world", "start": 0.52, "end": 1.1 }
  ]
}
200 Translated / cached 400 YouTube download failed 500 ffmpeg missing / Whisper error
GET /youtube/transcribe/{video_id}/sentences Get sentence-segmented transcript

Returns the cached transcription broken into natural sentences with start/end timestamps and per-word timing. Sentences are split on terminal punctuation (. ! ?) and words are aligned to the Whisper word-level output. Returns 404 if the video has not been transcribed yet.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID (e.g. dQw4w9WgXcQ)
Response · 200 OK
{
  "video_id": "dQw4w9WgXcQ",
  "language": "ko",
  "sentences": [
    {
      "text": "Hello, this is the first sentence.",
      "start": 0.0,
      "end": 2.5,
      "words": [
        { "word": "Hello,", "start": 0.0, "end": 0.48 }
      ]
    }
  ]
}
200 Sentences returned 404 Video not yet transcribed

Videos Domain

Retrieve stored video records with their associated subtitle tracks

The Videos domain exposes read access to the videos table. A video record is created automatically when a YouTube translation is first processed. The response includes the full list of associated subtitle tracks as a nested array.

GET /videos/{video_id} Fetch a video by ID with its subtitle tracks

Returns the stored video record identified by its YouTube video ID. The response includes metadata (title, duration, source URL, default language) and all associated subtitle tracks. Returns 404 if the video has not been processed.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID (e.g. dQw4w9WgXcQ)
Response · 200 OK
{
  "id": "dQw4w9WgXcQ",
  "title": "Video Title",
  "duration_ms": 212000,
  "source_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
  "default_lang": "en",
  "created_at": "2026-05-13T10:00:00",
  "updated_at": "2026-05-13T10:00:00",
  "subtitle_tracks": [
    {
      "language_code": "ko",
      "file_url": "http://localhost:8000/files/dQw4w9WgXcQ/ko.vtt",
      "format": "vtt"
    }
  ]
}
200 Video found 404 Video not found

Subtitles Domain

Upload, retrieve, and delete subtitle tracks (VTT/SRT) per video and language

The Subtitles domain manages subtitle files stored in the configured storage backend (local filesystem or S3). Each subtitle track is uniquely identified by its video_id + language_code pair. Files are stored under a key derived from the video ID and language code, and the public URL is persisted in the database. Retrieval issues a 302 redirect directly to the file URL.

POST /videos/{video_id}/subtitles Upload a subtitle track

Accepts a multipart/form-data upload. Stores the subtitle file in the configured storage backend, persists the track record in the database, and returns the created track metadata. Uploading for an existing language replaces the track.

Form fields
FieldTypeDescription
file requiredfileSubtitle file content (VTT or SRT)
language_code requiredstringBCP-47 language code, e.g. en, ko, ja
format optionalstringFile format: vtt (default) or srt
Example request
# curl multipart upload
curl -X POST http://localhost:8000/videos/dQw4w9WgXcQ/subtitles \
  -F "file=@subtitles_ko.vtt" \
  -F "language_code=ko" \
  -F "format=vtt"
Response · 200 OK
{
  "id": 1,
  "video_id": "dQw4w9WgXcQ",
  "language_code": "ko",
  "format": "vtt",
  "file_url": "http://localhost:8000/files/dQw4w9WgXcQ/ko.vtt",
  "file_size_bytes": 4096,
  "is_machine_translated": false,
  "status": "published",
  "created_at": "2026-05-13T10:00:00",
  "updated_at": "2026-05-13T10:00:00"
}
200 Track created / replaced 404 Video not found
GET /videos/{video_id}/subtitles/{language_code} Redirect to subtitle file

Looks up the subtitle track for the given video and language, then issues a 302 redirect to the file's public URL. The client follows the redirect and downloads the subtitle file directly from storage. Returns 404 if no track exists for that combination.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID
language_code requiredstringLanguage code matching a stored track
302 Redirect to file URL 404 Track not found
DELETE /videos/{video_id}/subtitles/{language_code} Delete a subtitle track

Deletes both the stored subtitle file from the storage backend and the corresponding database record. Idempotent — if the file does not exist in storage the record is still removed. Returns 204 No Content on success.

Path parameters
ParameterTypeDescription
video_id requiredstringYouTube video ID
language_code requiredstringLanguage code of the track to delete
204 Deleted 404 Track not found

Health

Liveness probe for monitoring and load balancers

GET /health Liveness check

Returns a static {"status": "ok"} JSON response. No database or external service check is performed — this is a pure liveness probe.

Response · 200 OK
{ "status": "ok" }
200 Service is up
